# 飞书应用权限配置快速指南

## 国际版 (larksuite.com)

### 1. 确认应用区域
✅ 应用必须在 https://open.larksuite.com/ 创建
❌ 不能在 https://open.feishu.cn/ 创建

### 2. 添加权限
进入应用 → 权限管理 → 添加权限：
- 搜索: `docx:document:readonly`
- 或搜索: `查看云文档内容`
- 添加并保存

### 3. 启用测试版本（推荐，无需审批）

1. 进入应用 → 左侧菜单点击 **"发布管理"**
2. 找到 **"测试版本"** 卡片
3. 点击 **"启用"**
4. 添加测试用户：
   - 输入你的邮箱
   - 点击发送邀请
   - 接受邀请邮件

⚡ 优势：无需管理员审批，立即可用！

### 4. 配置重定向 URL

进入应用 → **"设置与发布"** → **"重定向 URL"**：

添加：
```
https://forlark.zeabur.app/callback.html
```

### 5. 获取凭证

进入应用 → **"凭证与基础信息"**：
- 复制 **App ID**
- 复制 **App Secret**

### 6. 在插件中配置

1. 打开 Chrome 插件
2. 填写 App ID 和 App Secret
3. 点击"保存配置"
4. 点击"检查应用权限"验证
5. 测试获取文档

---

## 常见错误码

| 错误码 | 说明 | 解决方案 |
|--------|------|----------|
| 1770032 | 权限不足 | 添加权限并发布/启用测试 |
| 99991663 | 无权访问文档 | 使用用户授权或设置文档为公开 |
| 404 | 应用不存在 | 检查 App ID 是否正确 |
